ECHO - Sonar Sinyali Oyunu
============================

Designed by MSTYLDRM

Nasıl Oynanır:
- Sol buton (YUKARI): Sinyali yukarı hareket ettirir
- Sağ buton (AŞAĞI): Sinyali aşağı hareket ettirir
- Butonlara basılı tutarak sürekli hareket edebilirsiniz
- Yeşil engellerin arasındaki boşluklardan geçin
- Her 10 engelde zorluk artar

Mobil Cihazlarda Yükleme:
1. Safari/Chrome'da oyunu açın
2. "Paylaş" veya menüye tıklayın
3. "Ana Ekrana Ekle" seçeneğini seçin
4. ECHO simgesi ana ekranınıza eklenecektir

Dosyalar:
- index.html: Ana oyun dosyası
- favicon.ico: Tarayıcı simgesi
- apple-touch-icon-*.png: iPhone/iPad simgeleri
- maskable-icon-*.png: Android simgeleri
- manifest.json: PWA manifest dosyası

Teknolojiler:
- HTML5 Canvas
- Vanilla JavaScript
- CSS3
- Progressive Web App (PWA)
